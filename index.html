Atomic GEO Score ($S_{ij}$)

Computes specific performance of brand mentions for individual prompts on individual engines.

Inputs: M, R, T, I
TEXTBOOK MATHEMATICAL NOTATION
$$S_{ij} = M_{ij} \times R_{ij} \times T_{ij} \times I_i$$
PLAIN ENGLISH EXPLANATION
Prompt Score = Mention Weight $\times$ Rank Weight $\times$ Sentiment Weight $\times$ Intent Weight

Step 2: Cluster-Level Layer

Cluster Score ($CS_{lj}$)

Groups prompts by semantic cluster/business category to prevent volume-bias and assess intent segments.

Inputs: $S_{ij}$, $S_{\max}$
TEXTBOOK MATHEMATICAL NOTATION
$$CS_{lj} = \frac{\sum_{p_i \in P_l} S_{ij}}{n_l \times S_{\max}} \times 100$$
PLAIN ENGLISH EXPLANATION
Cluster Score = [ (Sum of Prompt Scores in Cluster) / (Number of Prompts $\times$ Max Possible Score) ] $\times$ 100

Step 3: Engine-Level Weighting

Engine GEO Score ($EG_j$)

Consolidates different cluster performances into a single robust performance index per LLM engine.

Inputs: $CS_{lj}$, Cluster Weights ($W_l$)
TEXTBOOK MATHEMATICAL NOTATION
$$EG_j = \sum_{l=1}^{k} W_l \times CS_{lj}$$
PLAIN ENGLISH EXPLANATION
Engine GEO Score = Sum of (Cluster Weight $\times$ Cluster Score) for all Clusters

Step 4: Unified GEO Layer

Unified Global Score ($GEO$)

A singular benchmark representing brand presence across the entire generative engine ecosystem.

Inputs: $EG_j$, Platform Shares ($A_j$)
TEXTBOOK MATHEMATICAL NOTATION
$$GEO = \sum_{j=1}^{m} A_j \times EG_j$$
PLAIN ENGLISH EXPLANATION
Global GEO Score = Sum of (Platform Weight $\times$ Engine GEO Score) across all Engines

Python Class Implementation

Production-grade, highly-commented Python class. Designed to prevent calculation inaccuracies (e.g., absent rank checks, zero divisors, validation bounds). Supports strict type-hinting. 

"""
GEO Framework Metric Engine
Standard mathematical implementation avoiding LaTeX parser dependency.
"""

class GEOScoringEngine:
    def __init__(self, m_max: float = 5.0, i_max: float = 2.0):
        self.M_MAX = m_max  # Theoretical max mention value
        self.I_MAX = i_max  # Theoretical max intent priority
        self.S_MAX = self.M_MAX * self.I_MAX  # Expected Max: 10.0

    def calculate_prompt_score(self, mention_weight: float, rank: int, sentiment_weight: float, intent_weight: float) -> float:
        """
        Step 1: Compute Atomic Prompt Level Score
        Formula: S_ij = M_ij * R_ij * T_ij * I_i
        """
        # Rank weight: If absent or invalid, rank factor is 0, else 1/rank
        rank_weight = 1.0 / rank if (rank and rank > 0) else 0.0
        
        # Ensure inputs lie in theoretical bounds
        clean_mention = min(max(mention_weight, 0.0), self.M_MAX)
        clean_sentiment = min(max(sentiment_weight, 0.0), 1.0)
        clean_intent = min(max(intent_weight, 0.0), self.I_MAX)

        return clean_mention * rank_weight * clean_sentiment * clean_intent

    def calculate_cluster_score(self, prompts_in_cluster: list, engine_id: str) -> float:
        """
        Step 2: Calculate normalized Cluster-Level Score for an Engine
        Formula: CS_lj = (Sum S_ij / (n_l * S_max)) * 100
        """
        if not prompts_in_cluster:
            return 0.0
        
        sum_prompt_scores = 0.0
        for prompt in prompts_in_cluster:
            mentions = prompt.get("mentions", {})
            ranks = prompt.get("ranks", {})
            sentiments = prompt.get("sentiments", {})
            
            s_ij = self.calculate_prompt_score(
                mention_weight=float(mentions.get(engine_id, 0.0)),
                rank=int(ranks.get(engine_id, 0)),
                sentiment_weight=float(sentiments.get(engine_id, 0.0)),
                intent_weight=float(prompt.get("intentWeight", 0.0))
            )
            sum_prompt_scores += s_ij

        prompt_count = len(prompts_in_cluster)
        return (sum_prompt_scores / (prompt_count * self.S_MAX)) * 100.0

    def calculate_engine_score(self, clusters: dict, cluster_weights: dict, engine_id: str) -> float:
        """
        Step 3: Calculate Engine Level score
        Formula: EG_j = Sum( W_l * CS_lj )
        """
        engine_score = 0.0
        
        # Ensure cluster weights are normalized (sum to 1)
        weight_sum = sum(cluster_weights.values())
        if weight_sum == 0:
            return 0.0
            
        for cluster_id, prompts in clusters.items():
            raw_weight = cluster_weights.get(cluster_id, 0.0)
            normalized_weight = raw_weight / weight_sum
            
            cluster_score = self.calculate_cluster_score(prompts, engine_id)
            engine_score += normalized_weight * cluster_score
            
        return engine_score

    def calculate_global_geo(self, engines: list, clusters: dict, cluster_weights: dict, platform_weights: dict) -> float:
        """
        Step 4: Calculate final overall GEO Score
        Formula: GEO = Sum( A_j * EG_j )
        """
        global_geo = 0.0
        platform_weight_sum = sum(platform_weights.values())
        if platform_weight_sum == 0:
            return 0.0

        for engine_id in engines:
            raw_platform_weight = platform_weights.get(engine_id, 0.0)
            normalized_platform_weight = raw_platform_weight / platform_weight_sum
            
            engine_score = self.calculate_engine_score(clusters, cluster_weights, engine_id)
            global_geo += normalized_platform_weight * engine_score
            
        return global_geo
